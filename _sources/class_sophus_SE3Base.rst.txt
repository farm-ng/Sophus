Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

:ref:`SE3 <doxid-classsophus_1_1_s_e3>` base type - implements :ref:`SE3 <doxid-classsophus_1_1_s_e3>` class but is storage agnostic.

SE(3) is the group of rotations and translation in 3d. It is the semi-direct product of SO(3) and the 3d Euclidean vector space. The class is represented using a composition of :ref:`SO3 <doxid-classsophus_1_1_s_o3>` for rotation and a one 3-vector for translation.

SE(3) is neither compact, nor a commutative group.

See :ref:`SO3 <doxid-classsophus_1_1_s_o3>` for more details of the rotation representation in 3d.

Logarithmic map.

Computes the logarithm, the inverse of the group exponential which maps element of the group (rigid body transformations) to elements of the tangent space (twist).

To be specific, this function computes ``vee(logmat(.))`` with ``logmat(.)`` being the matrix logarithm and ``vee(.)`` the vee-operator of SE(3).
