Group exponential.

This functions takes in an element of tangent space (= rotation 3-vector plus logarithm of scale) and returns the corresponding element of the group :ref:`RxSO3 <doxid-classsophus_1_1_rx_s_o3>`.

To be more specific, thixs function computes ``expmat(hat(omega))`` with ``expmat(.)`` being the matrix exponential and ``hat(.)`` being the :ref:`hat() <doxid-classsophus_1_1_rx_s_o3_1a3e986742b6fddb6b6e9822d2c50fcc48>` -operator of RSO3.

	static :ref:`Transformation<doxid-classsophus_1_1_rx_s_o3_1a07829c2aebfb03e7ec513036aec2c8c2>` generator(int i)

Returns the ith infinitesimal generators of ``R+ x SO(3)``.

The infinitesimal generators of :ref:`RxSO3 <doxid-classsophus_1_1_rx_s_o3>` are:

.. ref-code-block:: cpp

	      |  0  0  0 |
	G_0 = |  0  0 -1 |
	      |  0  1  0 |
	
	      |  0  0  1 |
	G_1 = |  0  0  0 |
	      | -1  0  0 |
	
	      |  0 -1  0 |
	G_2 = |  1  0  0 |
	      |  0  0  0 |
	
	      |  1  0  0 |
	G_2 = |  0  1  0 |
	      |  0  0  1 |

Precondition: ``i`` must be 0, 1, 2 or 3.

	static :ref:`Transformation<doxid-classsophus_1_1_rx_s_o3_1a07829c2aebfb03e7ec513036aec2c8c2>` hat(:ref:`Tangent<doxid-classsophus_1_1_rx_s_o3_1a14e67db45fcc7da7eead998c460aba02>` const& a)

hat-operator

It takes in the 4-vector representation ``a`` (= rotation vector plus logarithm of scale) and returns the corresponding matrix representation of Lie algebra element.

Formally, the :ref:`hat() <doxid-classsophus_1_1_rx_s_o3_1a3e986742b6fddb6b6e9822d2c50fcc48>` -operator of :ref:`RxSO3 <doxid-classsophus_1_1_rx_s_o3>` is defined as

``hat(.): R^4 -> R^{3x3}, hat(a) = sum_i a_i * G_i`` (for i=0,1,2,3)

with ``G_i`` being the ith infinitesimal generator of :ref:`RxSO3 <doxid-classsophus_1_1_rx_s_o3>`.

The corresponding inverse is the :ref:`vee() <doxid-classsophus_1_1_rx_s_o3_1a6ce6cdf2cf8d19ae17a045407bea6896>` -operator, see below.

Lie bracket.

It computes the Lie bracket of RxSO(3). To be more specific, it computes

``[omega_1, omega_2]_rxso3 := vee([hat(omega_1), hat(omega_2)])``

with ``[A,B] := AB-BA`` being the matrix commutator, ``hat(.)`` the :ref:`hat() <doxid-classsophus_1_1_rx_s_o3_1a3e986742b6fddb6b6e9822d2c50fcc48>` -operator and ``vee(.)`` the :ref:`vee() <doxid-classsophus_1_1_rx_s_o3_1a6ce6cdf2cf8d19ae17a045407bea6896>` -operator of :ref:`RxSO3 <doxid-classsophus_1_1_rx_s_o3>`.

vee-operator

It takes the 3x3-matrix representation ``Omega`` and maps it to the corresponding vector representation of Lie algebra.

This is the inverse of the :ref:`hat() <doxid-classsophus_1_1_rx_s_o3_1a3e986742b6fddb6b6e9822d2c50fcc48>` -operator, see above.

Precondition: ``Omega`` must have the following structure:

.. code-block:: cpp

	|  d -c  b |
	|  c  d -a |
	| -b  a  d |
