Group exponential.

This functions takes in an element of tangent space (= rotation angle plus logarithm of scale) and returns the corresponding element of the group :ref:`RxSO2 <doxid-classsophus_1_1_rx_s_o2>`.

To be more specific, this function computes ``expmat(hat(theta))`` with ``expmat(.)`` being the matrix exponential and ``hat(.)`` being the :ref:`hat() <doxid-classsophus_1_1_rx_s_o2_1aae5c96c5e130a43beb4eba3320e5f205>` -operator of RSO2.

	static :ref:`Transformation<doxid-classsophus_1_1_rx_s_o2_1a85b426eeb268d2d550da9fa86a066b95>` generator(int i)

Returns the ith infinitesimal generators of ``R+ x SO(2)``.

The infinitesimal generators of :ref:`RxSO2 <doxid-classsophus_1_1_rx_s_o2>` are:

.. ref-code-block:: cpp

	      |  0 -1 |
	G_0 = |  1  0 |
	
	      |  1  0 |
	G_1 = |  0  1 |

Precondition: ``i`` must be 0, or 1.

hat-operator

It takes in the 2-vector representation ``a`` (= rotation angle plus logarithm of scale) and returns the corresponding matrix representation of Lie algebra element.

Formally, the :ref:`hat() <doxid-classsophus_1_1_rx_s_o2_1aae5c96c5e130a43beb4eba3320e5f205>` -operator of :ref:`RxSO2 <doxid-classsophus_1_1_rx_s_o2>` is defined as

``hat(.): R^2 -> R^{2x2}, hat(a) = sum_i a_i * G_i`` (for i=0,1,2)

with ``G_i`` being the ith infinitesimal generator of :ref:`RxSO2 <doxid-classsophus_1_1_rx_s_o2>`.

The corresponding inverse is the :ref:`vee() <doxid-classsophus_1_1_rx_s_o2_1a4478849aa13153d0578f6afe2b4a3e20>` -operator, see below.

Lie bracket.

It computes the Lie bracket of RxSO(2). To be more specific, it computes

``[omega_1, omega_2]_rxso2 := vee([hat(omega_1), hat(omega_2)])``

with ``[A,B] := AB-BA`` being the matrix commutator, ``hat(.)`` the :ref:`hat() <doxid-classsophus_1_1_rx_s_o2_1aae5c96c5e130a43beb4eba3320e5f205>` -operator and ``vee(.)`` the :ref:`vee() <doxid-classsophus_1_1_rx_s_o2_1a4478849aa13153d0578f6afe2b4a3e20>` -operator of :ref:`RxSO2 <doxid-classsophus_1_1_rx_s_o2>`.

vee-operator

It takes the 2x2-matrix representation ``Omega`` and maps it to the corresponding vector representation of Lie algebra.

This is the inverse of the :ref:`hat() <doxid-classsophus_1_1_rx_s_o2_1aae5c96c5e130a43beb4eba3320e5f205>` -operator, see above.

Precondition: ``Omega`` must have the following structure:

.. code-block:: cpp

	|  d -x |
	|  x  d |
